# Freelancer Project Dashboard

A complete Notion template for freelancers to manage multiple projects, track invoices, monitor deadlines, and organize client communications. Includes: project database, invoice tracker, client CRM, weekly planner, and goal setting pages. Perfect for independent designers, developers, and consultants.

## What's Included
  - Project database with status tracking
  - Invoice tracker with payment status
  - Client CRM with contact info
  - Weekly planner view
  - Goal setting dashboard
  - Time tracking log

Notion Template
